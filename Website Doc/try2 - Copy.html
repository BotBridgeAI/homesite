Expert Section

This section contains information on expert features/settings for expert users.

Expert Settings Section

These settings can be modified if you want the bot to respond in a certain way.

Previous Sentences Window

This is the number of sentences directly before the last bot message sent that should be considered during the message generation.

Temporal Sentence Sampling

This is an additonal way for the bot to gain context about what has already happened, it is more vague but includes a wider variety of context from previous messages. It can be used in cadence with the Previous Sentence Window as it will ignore any selected within this window. This setting works by weighting all the sentences by how recent the content was said and randomly selects sentences from the entire chats. Newer sentences will have higher weights are are more likely to be selected. After the sentences are selected they are then ordered by weight to keep the timeline roughtly in-line.

API Security

For securing the API, ensure that you are using proper authentication methods such as JWT (JSON Web Token) or OAuth.

Example of a secure request:

curl -H "Authorization: Bearer your-token" https://api.yoursite.com/protected-endpoint

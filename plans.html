Free Connection Plan

Our Free Connection Plan allows users to access our AI chatbot services without any fees. Here are the details:

  • API Link: Access via an API link with an auth token.
  • Requirements to Enroll:
    • AI functions cannot be restricted to paid use. However, users can shave up to 8 seconds off their response time by using our priority function (1 to 5) with a donation of API resources.
    • We manage ads on the chatting page but limit their number generously. Ads are removed for users who donate resources.
    • You may charge users to remove ads; however, we request a minimum of $3 per user for this service to help sustain our free offerings.
    • No commissions will be charged on non-AI-related activities on your platform, only for services that utilize our system.
    • Priority can be obtained by donating to BotBridgeAI; you will receive 70% if the user indicates they came from your platform.
    • Encouragement for users to host an API is required.

BotBridge ProfitEdge Plan

The BotBridge ProfitEdge Plan is designed for businesses that want more control over their AI services. Here are the details:

    Pricing

    We offer a tiered pricing structure to accommodate various usage levels:

    • Tier 0: $0.0010 per API call for the first 25,000 requests in a month (~100 users x 250). This tier is offered to help you get your business started.
    • Tier 1: $0.0015 per API call for the first 1,000,000 requests/month. (~4000 users x 250)
    • Tier 2: $0.0012 per API call for 1,000,001 to 3,000,000 requests/month. (~12000 users x 250)
    • Tier 3: $0.0011 per API call for 10,000,001 to 50,000,000 requests/month. (~40000 users x 250)
    • Tier 4: $0.0010 per API call for 50,000,001+ requests/month. (~200000 users x 250)
  • API Link: Access via an API link with an auth token.
  • Benefits:
    • No ads from our end.
    • Ability to charge for AI services on your site.
    • More secure communication channels.
    • Higher priority support.

Additional Costs

For either plan, the following additional costs apply:

  • Self-Implementation: $5 (one-time fee for setup).
  • AI Implementation: $500 - $2500 (depending on the complexity of the integration).
